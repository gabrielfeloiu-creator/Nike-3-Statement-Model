# Nike 3-Statement Financial Model
Built a fully linked 3-statement financial model for Nike (NKE) from scratch as a self-directed project to develop financial modeling skills ahead of finance recruiting.
What I built
Income statement, balance sheet, and cash flow statement using 3 years of historical data pulled directly from Nike's FY2025 10-K filing on SEC EDGAR. All three statements are fully linked — changes in revenue flow through to net income, retained earnings, and ending cash automatically.
Added 5 years of projections (FY2026-FY2030) driven by a separate assumptions tab. Revenue growth anchored to Yahoo Finance analyst consensus for near-term years, with long-run estimates from published 5-year growth forecasts.
